Решение задач из паскаля на языках программирования Python и Javascript

Условие

Дан диаметр окружность d. Найти её длину L = π*d. В качестве значения π использовать 3.14.

Примеры

Исходные данные
Пример верного решения
d = 1
L = 3.14
d = 7
L = 21,98

Решение

Решение на языке Python

	d = int(input("Введите диаметр окружности: "))
	P = 3.14
	print("L = ",(d*P))

Решение на языке Javascript

	const pi = 3.14;
	var d = +prompt("d = ");
	var L = pi*d;
	alert("L = " + L);

Комментарий

При решении задачи на языке Python при возведении в степень можно использовать **2, а можно подключить библиотеку math и использовать math.pow(a,2)
