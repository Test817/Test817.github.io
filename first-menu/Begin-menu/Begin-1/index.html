Решение задач из паскаля на языках программирования Python и Javascript

Условие

Дана сторона квадрата. Найти его периметр P = 4*a

Примеры

Исходные данные
Пример верного решения
a = 1
P = 4
a = 1.7
P = 6.8

Решение

Решение на языке Python

	a = int(input("Введите сторону квадрата: "))
	print("P =",(a*4))

Решение на языке Javascript

	var a = +prompt("a = ");
	var P = Math.round(4*a*10) / 10;
	alert("S = " + P);
