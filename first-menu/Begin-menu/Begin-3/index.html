Решение задач из паскаля на языках программирования Python и Javascript

Условие

Даны стороны прямоугольника a и b. Найти его площадь S = a*b и периметр P = 2*(a + b).

Примеры

Исходные данные
Пример верного решения
a = 3
b = 4
S = 12
P = 14
a = 9.8
b = 2.5
S = 24.5
P = 24.6

Решение

Решение на языке Python

	a = int(input("Длина: "))
	b = int(input("Ширина: "))
	print("P = {0} , S = {1}".format(2*a+2*b,a*b))

Решение на языке Javascript

	var a = +prompt("a = ");
	var b = +prompt("b = ");
	var S = a*b;
	var P = 2*(a+b);
	alert("S = " + S);
	alert("P = " + P);
