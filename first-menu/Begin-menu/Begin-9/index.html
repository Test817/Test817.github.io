Решение задач из паскаля на языках программирования Python и Javascript

Условие

Даны два неотрицательных числа a и b. Найти их &nbs среднее геометрическое, т.е квадратный корень из их произведения: Sqrt(a*b).

Примеры

Исходные данные
Пример верного решения
a = 4
b = 9
6

Решение

Решение на языке Python

import math
a = int(input("Введите первое число: "))
b = int(input("Введите второе число: "))
print(math.sqrt(a*b))

Решение на языке Javascript

var a = +prompt("a = ");
var b = +prompt("b = ");
var srg = Math.sqrt(a * b);
srg = Math.round(srg * 100) / 100;
alert(srg)

Комментарий

В решении используется библиотека math.

Комментарий

Сначала, с помощью метода Math.sqrt(), извлекается квадратный корень из числа a*b, а затем полученное число округляется до сотых с помощью метода Math.round().
