Решение задач из паскаля на языках программирования Python и Javascript

Условие

Найти длину окружности L и площадь круга S заданного радиуса R: L = 2*π*R, S = π*R. В качестве значения π использовать 3.14.

Примеры

Исходные данные
Пример верного решения
R = 6.5
L = 40.82
S = 132.665

Решение

Решение на языке Python

	r = int(input("Введите радиус окружности: "))
	P = 3.14
	print("L = {0},S = {1}".format(2*P*r,P*r**2))

Решение на языке Javascript

	const pi = 3.14;
	var r = +prompt("R = ");
	var L = 2*pi*r;
	var S = pi*Math.pow(r,2);
	alert("L = " + L + "        S = " + S);
