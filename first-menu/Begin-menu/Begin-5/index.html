Решение задач из паскаля на языках программирования Python и Javascript

Условие

Дана длина ребра куба a. Найти обьем куба V = a³ и площадь его поверхности S = 6*a².

Примеры

Исходные данные
Пример верного решения
a = 2
V = 8
S = 24

Решение

Решение на языке Python

a = int(input("Введите длину ребра куба: "))
print("V = {0},S = {1}".format(a**3,6*a**2))

Решение на языке Javascript

var a = +prompt("a = ");
var V = Math.pow(a,3);
var S = 6*Math.pow(a,2);
alert("V = " + V + "        S = " + S);
