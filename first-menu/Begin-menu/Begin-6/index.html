Решение задач из паскаля на языках программирования Python и Javascript

Условие

Дан диаметр окружности d. Найти её длину L = π*d. В качестве значения π использовать 3.14.

Примеры

Исходные данные
Пример верного решения
d = 2
L = 6.28

Решение

Решение на языке Python

d = int(input("Введите диаметр окружности: "))
P = 3.14
print("L = ",(d*P))

Решение на языке Javascript

const pi = 3.14;
var d = +prompt("d = ");
var L = pi*d;
alert("L = " + L);
